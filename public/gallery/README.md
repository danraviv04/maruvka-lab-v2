# Gallery Images

This folder contains images for the lab gallery page.

## How to add images

1. Place your images in this folder (`public/gallery/`)
2. Update `content/gallery.ts` to reference your images
3. Image paths should be `/gallery/filename.jpg`

## Recommended specifications

- **Format**: JPG or PNG
- **Size**: 1200px wide (or larger for high-DPI displays)
- **Aspect ratio**: 4:3 works well for the gallery grid
- **File size**: Optimize images to < 500KB when possible

## Image categories

- `lab`: Lab space, equipment, workspace photos
- `research`: Scientific figures, plots, visualizations
- `events`: Team photos, conferences, seminars, retreats
- `figures`: Published figures, journal covers

## Example filenames

- `lab-space.jpg`
- `team-meeting.jpg`
- `signatures-plot.jpg`
- `conference-2024.jpg`
- `retreat-2024.jpg`

Make sure to update the corresponding entries in `content/gallery.ts` with accurate titles and descriptions.
